# CalculadoraIMCApp
Proyecto .NET MAUI sencillo que calcula el IMC (Índice de Masa Corporal) usando MVVM con CommunityToolkit.Mvvm.

## Cómo usar
1. Abra la carpeta del proyecto en Visual Studio 2022/2023 que soporte .NET MAUI o Visual Studio 2022/2023 Preview según su instalación.
2. Restaure paquetes NuGet (Restore).
3. Seleccione la plataforma (Windows, Android, iOS) y ejecute.
## Comandos CLI (si tiene workloads MAUI instalados)
- dotnet build
- dotnet run -f:net8.0-windows10.0.19041.0 (o la TFM correspondiente)
